# Minkowski Space-Time Interaction 🚀
<img width="838" height="662" alt="ЕСССС_ФИЗИКА_ЮХУ" src="https://github.com/user-attachments/assets/0a2ec248-b381-4284-86b5-dc0bd683722c" />

Интерактивный симулятор двумерной диаграммы Минковского, разработанный в качестве индивидуального проекта для колледжа. Программа визуализирует фундаментальные концепции Специальной Теории Относительности (СТО), такие как замедление времени, сокращение длины и относительность одновременности.

## 📖 О проекте

Проект представляет собой графическую среду, в которой пользователь может изменять относительную скорость наблюдателя и видеть, как искажается пространство-время с точки зрения разных инерциальных систем отсчета (ИСО).

### Основные функции:
*   **Двойная координатная сетка:** Неподвижная система ($S$) и движущаяся система ($S'$) отображаются одновременно.
*   **Релятивистский HUD:** Панель управления, отображающая текущую скорость ($\beta$), Лоренц-фактор ($\gamma$) и угол наклона осей в реальном времени.
*   **Визуализация объектов:** Группа квадратов-маркеров, расположенных по осям и диагоналям (световой конус), демонстрирует геометрические искажения Лоренца.
*   **Интерактивность:** Плавное изменение скорости клавишами управления.

## 🔬 Физико-математический аппарат

Программа базируется на преобразованиях Лоренца. Пространство и время рассматриваются как единое целое — пространство Минковского, где временная координата представлена как $ct$ (расстояние, проходимое светом за время $t$).

### Ключевые параметры:
1.  **Относительная скорость ($\beta$):** 
    $$\beta = \frac{v}{c}$$
2.  **Лоренц-фактор ($\gamma$):** определяет степень релятивистских эффектов.
    $$\gamma = \frac{1}{\sqrt{1 - \beta^2}}$$
3.  **Преобразование координат:**
    Для отрисовки объектов из системы $S'$ на экране системы $S$ используются обратные преобразования:
    $$x = \gamma (x' + \beta \cdot ct')$$
    $$ct = \gamma (ct' + \beta \cdot x')$$

## 📦 Установка и запуск

### Вариант 1: Запуск готовой сборки (Windows)
Если вы хотите просто запустить программу без компиляции:
1. Перейдите в папку `build/`.
2. Запустите файл `app.exe`.
   *   **Примечание:** Для работы могут потребоваться библиотеки `SDL3.dll`. Если они отсутствуют в папке, их нужно скачать с официального сайта SDL или взять из папки зависимостей проекта.

Если вы хотите скомпилировать проект самостоятельно:
1.  Установите библиотеку **SDL3**.

    git clone https://github.com/Leonard457/STO_DUO_Coledj_poekt.git
    ```
3.  Скомпилируйте проект с помощью вашего компилятора (g++, clang или MSVC), указав пути к заголовочным файлам и библиотекам SDL3.
    *Пример для g++:*

    ```

## 🎮 Управление

*   **Стрелка ВПРАВО (→):** Увеличить скорость (движение вперед по оси X).
*   **Стрелка ВЛЕВО (←):** Увеличить скорость в обратном направлении (движение назад по оси X).
*   **Диапазон скорости:** от $-0.99c$ до $0.99c$.

## 📂 Структура файлов

*   `main.cpp` — Основной цикл приложения, логика отрисовки сетки, фигур и HUD.
*   `physics/STO.h` — Математическое ядро проекта: расчет $\gamma$, $\beta$ и функций преобразования координат.
*   `build/` — Папка с скомпилированным исполняемым файлом `app.exe`.

## 📈 Результаты проекта
Программа наглядно демонстрирует, что при достижении околосветовых скоростей:
1.  Оси времени и пространства движущейся системы "складываются" по направлению к световому конусу (угол $45^\circ$).
2.  События, которые были квадратными в своей ИСО, для неподвижного наблюдателя выглядят как вытянутые параллелограммы (относительность одновременности).
3.  Объекты на световом конусе всегда остаются на нем, подтверждая инвариантность скорости света.
